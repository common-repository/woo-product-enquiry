== Description ==

A quick product enquiry form. Enquiry form will be added to product details page. Get emails about enquiries and very easy to customize.

View all your product enquiries from admin dashboard at a glance.

We support multi-vendors too. 

More managable settings is coming soon. For any further customization feel free to raise a ticket on our [website](http://www.mingocommerce.com/contact-us/)

== Installation ==

Manual

1. Upload the plugin folder to the '/wp-content/plugins/' directory .

2. Activate the plugin through the 'Plugins' menu in WordPress.

Automatic

1. Go to plugin list from 'dashboard'. Search our plugin and click to install

2. Activate it.

== Frequently Asked Questions ==
 
= Can I add new fields to  the form ? =
 
Yes, you can add/modify fields to enquiry form using hooks. Default field labels (e.g. name, email) can be customized through admin panel. 
 
= Can I add HTML to the form ? =
 
Yes, hooks are available for that.

= Is there any dashboard page for enquiries ? =

Yes, under products tab of dashboard, there is a page 'Product Enquiry', where you can find all enquiries.

= Does this work with multivendor plugin ? =

Yes, it does.

= Where can I get help for customizing ? =

Well, few things can be done through available hooks. If you are not familier with hooks/development you can [contact us](http://www.mingocommerce.com/contact-us/) regarding changes
